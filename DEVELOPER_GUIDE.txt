Add New API Endpoint:

File: server.js

Template:
```javascript
app.get('/api/your-endpoint', (req, res) => {
    try {
        // Your logic here
        res.json({ data: 'your response' });
    } catch (error) {
        console.error('Error:', error);
        res.status(500).json({ error: 'Error message' });
    }
});
```

Example - Get student count:
```javascript
app.get('/api/student-count', (req, res) => {
    try {
        const count = 1500; // Your data
        res.json({ students: count });
    } catch (error) {
        res.status(500).json({ error: 'Failed to fetch count' });
    }
});
```

Then use in JavaScript:
```javascript
fetch('/api/student-count')
    .then(res => res.json())
    .then(data => console.log(data.students));
```

ğŸ’¾ UPGRADE TO PERSISTENT DATABASE

Currently: Data stored in memory (lost on server restart)
Goal: Store in Supabase PostgreSQL

Steps:
1. In Supabase, create table "subjects":
   - id (primary key)
   - class (text)
   - name (text)
   - created_at (timestamp)

2. In server.js, replace array with database query:

Before:
  const filtered = subjectsData.filter(q => q.subject === subject);

After:
  const { data, error } = await supabase
    .from('subjects')
    .select()
    .eq('subject', subject);

3. Test and deploy!
